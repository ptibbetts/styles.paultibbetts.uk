About this website

This website acts as my central hub online. It's been created with an indieweb focus to help me own my own data.

It's built with WordPress using Trellis, Bedrock and Sage (all by the Roots team) with a whole bunch of plugins.

I've split the source code of it into three repositories so that I can concentrate on different parts of the website at different times:

  • The styleguide is available at https://ptibbetts.github.io/styles.paultibbetts.uk/
  • The backend code is available at https://github.com/ptibbetts/paultibbetts.uk
  • The frontend code is available at https://github.com/ptibbetts/allusion
